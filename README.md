## Установка

Установите гемы

```
bundle install
```

## Запуск

Создайте базу данных

```
rails db:create
rails db:migrate
```

Заполните базу данных тестовыми данными:

```
rails db:seed
```

Запустите приложения командой:

```
bin/dev
```
